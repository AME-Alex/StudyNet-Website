Layer 3: The Network Layer

Introduction

The Network Layer is responsible for moving data between different networks using IP addresses. It determines the best path for data to travel across multiple networks and uses routers to forward packets toward their destination.

Routers

Routers connect different networks together. They are the primary Layer 3 device that handles routing.

  • Your home network → Internet
  • Office network → Another office network

How it Works

  1. Your computer wants to visit a website.
  2. It sees the website is not on the local network.
  3. It sends the packet to the default gateway (router).
  4. Routers move the packet across multiple networks until it reaches the destination.

Responsibilities

  • Logical addressing (IP)
  • Routing (finding best path)
  • Packet forwarding
  • Network separation (broadcast domains)

Why Layer 3 Matters

Without Layer 3 there is no Internet — no communication between networks and no routing.
