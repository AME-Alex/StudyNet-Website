Layer 1/2: The Physical & Data Link Layers

Responsibilities

Layer 1 and Layer 2 are responsible for moving raw bits and turning them into frames for communication inside a single physical local network.

  • Framing (organizing bits into frames)
  • MAC addressing
  • Error detection (CRC)
  • Switching (forwarding frames inside a LAN)

MAC Addresses

Each network device has a MAC address (physical hardware ID). This address is used to deliver frames within the local network.

Example: 00:1A:2B:3C:4D:5E

Switches

A switch is a Layer 2 device that connects devices on the same network. It sends data only to the devices it is meant for.

How it Works

  1. Each device has a MAC address.
  2. The switch learns which MAC address is on which port.
  3. When data arrives, it forwards to the correct port.
  4. If unknown, it floods the frame to all ports (once).

Why Switches are Good

  • Faster than hubs
  • Reduces network traffic
  • Full-duplex support allows two devices to talk at the same time

Key Takeaways

Layer 2
